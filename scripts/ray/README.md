<h1 align="center" id="title">Deploy Autoscaling Ray Cluster with KubeRay in Kubernetes Cluster</h1>

## Install KubeRay

```bash
./install-kuberay.sh
```

## Start Ray Cluster with Autoscaling

```bash
./start-ray-cluster.sh
```

## Delete Ray Cluster with Autoscaling

```bash
./delete-ray-cluster.sh
```

## Uninstall KubeRay

```bash
./uninstall-kuberay.sh
```
